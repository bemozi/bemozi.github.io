## What is Statim?

[Statim](https://bemozi.github.io) is a **fully auditable web app** offers sustained functionality without connectivity and **complete data ownership**, all with a responsive, intuitive, and **distraction-free interface**.

Visit [Statim](https://bemozi.github.io) Homepage for more details at [https://bemozi.github.io](https://bemozi.github.io).

> [!NOTE]
> This repository only uses native, well-documented, and officially supported browser APIs without relying on any third-party dependencies.

## Development Status

This version is currently under active development.

> [!WARNING] Disclaimer
> The current version may contain bugs and unexpected behavior, and **data backups are mandatory** when working with sensitive information.
